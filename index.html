CSS Aesthetics

.alpha {

background-color: rgba(255,255,255,0.8);

}

Create a semi transparent background.

.drop-shadow {

box-shadow: 0px 3px 5px #444;

}

Create a drop shadow effect. {horizontal offset, vertical offset, blur, spread, color}

.linear-gradient {

background: linear-gradient (red, to top right, blue);

}

Create a linear gradient. You can specify the linear gradient direction, for example top to bottom means the heads to the top right starting with red and ending in blue. Direction can also be specified in degrees, i.e. 45deg.

CSS Media Queries

@media (min-width: 500px) { ... }

Styles in this block will only apply if the device width is >= 500px

@media (max-width: 750px) { ... }

Styles in this block will only apply if the device width is <= 750px

@media (max-width: 750px) and (min-width: 500px) { ... }

Styles in this block will only apply is the device width is between 500px-750px

CSS Flexbox

.flexbox-wrapper {

display: flex;

}

Set the wrapper element to be a flex box.

HTML Semantics

Sectioning Elements

<main>

Main content area of a document, should be unique.

<section>

Generic section of a document.

<nav>

Wrap major navigational links.

<article>

Wrap self-contained pieces of content.

<aside>

Content that is slightly related to the main content.

<header>

"Masthead" area, introduction to other content.

<footer>

Footer area.

Grouping Elements

<div>

Conveys nothing but meaning is implied when it's used to group something.

<p>

Markup a paragraph and should be used for any text.

<figure> & <figcaption>

Used to annotate illustrations, diagrams, photos, code listings, etc.
